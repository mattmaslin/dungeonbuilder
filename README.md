# dungeonbuilder

Based on the algorithm described [here]http://www.polygonpi.com/?p=1191)

![Screenshot](http://i.imgur.com/ej8TU1t.png)

## Example

```bash
git clone https://github.com/mattmaslin/dungeonbuilder 
cd dungeonbuilder
cargo run --features window --example builder
```
